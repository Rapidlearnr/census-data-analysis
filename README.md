# Census Data Analysis

This project analyzes census data using Python to study rural and urban population patterns.

## Objectives
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Missing Value Handling
- Linear Regression
- Hypothesis Testing

## Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn
- SciPy

## Description
The project performs preprocessing, visualization, correlation analysis, regression modeling, and hypothesis testing on district-level census data. Log transformation and outlier handling are used to improve analysis quality.

## How to Run
```bash
python analysis.py
